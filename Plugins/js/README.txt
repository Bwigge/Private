The files in this folder contain skeleton builds for each AT&T JavaScript file.

Files are named att-<platfrom>.js

Run the following command to build a att-<platfrom>-min.js to the build folder.



To Build using require.js module:
	1. Ensure node is installed
	2. Downloaded r.js from https://github.com/jrburke/r.js
	2. Create a JS build file for your project.  Follow the patterns at https://github.com/jrburke/r.js/blob/master/build/example.build.js
	4. Run the following command:
		r.js -o <build>.js


Reference for require.js Optimizer: http://requirejs.org/docs/optimization.html
